Acerca

Aplicación Android que ofrece una alternativa para consultar notas. Facilita la consulta de notas de los estudiantes de la Universidad Mariano Gálvez, utilizando la plataforma virtual proporcionada por la universidad

Demostración

Inicio de sesións

Utilizando la cuenta proporcionada por la universidad para ingresar a la plataforma virtual se llevará a cabo todo el proceso

Selección de notas

Se eligen las notas que se desean visualizar o simplemente se cargan las actuales

Visualización

Visualización de notas con posibilidad de verlas de dos formas, como se muestra en la imagen y en modo simple (Menú -> Vista simple)

¿Cómo funciona?

La aplicación se conecta a la plataforma virtuaal de la Universidad Mariano Gálvez para poder automatizar el proceso de consulta de notas del estudiante de una forma rápida y sencilla.

Es dependiente del sitio web y si este no está en línea o hay problemas con el mismo la aplicación no funcionara correctamente, como por ejemplo problemas de servidor, cola etc. Si tiene errores al conectarse o cargar notas en la aplicación asegúrese de que pueda hacerlo en el sitio web de la Universidad Mariano Gálvez

Inicio de sesión

Se iniciara la sesión con el usuario del estudiante y es posible recordar al usuario con el propósito de pedir credenciales una única vez

Selección de notas

Como en el sitio web, el estudiante tiene la opción de elegir cualquier Clase/Unidad/Semestre para visualizar las notas así también como el año

Visualización

Se muestran las notas seleccionadas por el estudiante, presentadas por curso también mostrando la cantidad de puntos necesarios para aprobar el curso
